在此基础上，经过 6 个场景、40+ 轮实测迭代，总结出"黄金三段式"提示词结构与年代分级系统，形成本 Skill 的完整方法论。转载或二次创作请注明来源。

## 适用模型

本技能的提示词方法论适用于任何AI图片生成模型，包括但不限于：豆包、Midjourney、DALL-E、Stable Diffusion、Flux、文心一格等。核心原理（设备+拍摄逻辑+画质缺陷+反AI否定）是通用的，不依赖特定模型的特性。

## 核心原理

直接写"真实"、"写实"、"照片级"只会触发一种AI风格。要生成真正有真实感的图片，需要用**具体的、可感知的不完美特征**来引导模型，让它以为你在描述一张真实存在的照片。

> **真实感 = 具体设备 + 拍摄逻辑 + 画质缺陷 + 不完美细节 + 普通人物 + 反AI否定**

**⚠️ 最重要的一条：设备名称不是装饰词！**

写"索尼Cyber-shot"不等于真实感——AI会把它当成背景板，然后继续用俯视大角度、完美构图、精致光影来画。**设备名称必须搭配那个年代/设备真实的拍摄逻辑**：

| 设备 | 真实拍摄逻辑 | AI默认会犯的错 |
|------|------------|--------------|
| 2000年山寨手机 | 单手举着按、画面歪、分辨率极低 | 俯视大角度、人物居中、画面清晰 |
| 2000年卡片机 | 平视随手按、构图歪、不等人 | 俯视大角度、人物居中、完美瞬间 |
| 70年代照相馆 | 师傅摆拍、正面闪光灯、布幕背景 | 艺术渐变背景、柔光、自然表情 |
| 老手机街拍 | 单手持机、画面歪、前景误入 | 专业构图、干净画面、精准对焦 |
| 胶卷相机 | 颗粒重、偏色、对焦不准 | 高清锐利、色彩精准、景深完美 |

**所以第一段必须同时写：用什么设备 + 用这个设备的人会怎么拍**。只写设备名不写拍摄逻辑，等于白写。

**关键发现**：
- 说"模糊"容易生成像素风 → 改用"分辨率不高"、"细节软软的"、"对焦不太准"
- 说"颗粒感"容易像油画 → 改用"噪点多"、"胶卷颗粒"、"扫描件质感"
- 人物必须强调"每个人长得都不一样" → 否则AI会用模板脸
- 年代越久远，泛黄、褪色、模糊的程度要逐级递增
- 墙面、建筑必须强调"斑驳、不规整、有岁月痕迹" → 否则AI会画得像3D建模
- 多人场景必须强调"所有人朝向一致" → 否则AI会把人物朝向搞乱
- **只写设备名不写拍摄逻辑 → AI会忽略年代约束，用默认的完美构图** → 必须明确写"随手举起相机按的快门"、"构图很烂"、"师傅硬摆的"等
- **说"饱和度低""褪色"模型会打折执行** → 改用极端重复表达："饱和度极低极低""灰扑扑黄乎乎的""没有任何一个颜色是鲜艳的""所有颜色都蒙着灰和黄"，语气要夸张到位，单次强调不够就反复说

## 黄金三段式提示词结构

经过6个场景、40+轮测试验证的高效结构，按顺序来：

### 第一段（置顶，权重最高）：年代 + 设备 + 拍摄逻辑 + 色彩 + 画质
用**具体的年代** + **具体的设备名** + **用这个设备的人会怎么拍** + **具体的色彩/画质描述**，直接定死整体风格。这一段决定了80%的最终效果。

**⚠️ 拍摄逻辑必须写！** 只写设备名不写"用这个设备的人会怎么拍"，AI会忽略年代约束，用默认的完美构图。每个年代的拍摄习惯完全不同，这是决定画面是否真实的关键。

✅ 好的开头（都包含了拍摄逻辑）：
- "70年代老照片，海鸥120双反相机在照相馆拍的，师傅让站好别动，正面闪光灯直射，表情僵硬不自然，严重褪色泛黄，饱和度极低，画质很差，分辨率不高，有很多噪点"
- "90年代彩色老照片，柯达金胶卷傻瓜机拍的，举起来就按快门没讲究构图，画面有点歪，褪色很严重，颜色整体偏黄偏暗，灰蒙蒙的像蒙了一层雾"
- "2000年索尼Cyber-shot卡片机随手拍的，单手举起来按的快门，构图很烂，画面略微倾斜，画质一般，分辨率不高，噪点很多，颜色发灰不鲜艳，偏色偏黄偏绿"
- "用手机随手拍的，没对好焦就按了，画面略微有点歪，构图很随意，画质一般，分辨率不高，有轻微数码噪点，白平衡不准有点偏色，色彩不鲜艳饱和度低"

❌ 坏的开头：
- "真实的照片"（太笼统，触发AI风格）
- "超写实人像"（还是AI词汇）
- "很模糊的照片"（容易生成像素风）
- "2000年索尼Cyber-shot拍的，画质一般"（❌ 只写了设备名，没写拍摄逻辑，AI会忽略年代继续用完美构图）

### 第二段（中间）：主体内容 + 不完美细节
描述画面主体，但要强调**普通、不完美、各有特点**，避免AI模板化。

- **人物**：普通人长相，不是网红脸，每个人长得都不一样，各有各的特点。眼睛不大，普通大小。表情自然不僵硬，有点拘谨有点呆。皮肤有毛孔有痘印。衣服有点旧有褶皱。
- **建筑/环境**：墙面斑驳，有脱落有水渍，脏脏旧旧的，不是规整的砖墙。招牌旧旧的，油漆掉了，没有鲜艳颜色。
- **构图**：有点歪，不在中心，有点裁切掉，随手拍的没讲究。
- **物品**：有点旧，有划痕，不是崭新完美的。
- **多人场景**：所有人朝向一致，绝对没有面向镜头的人，只能看到后脑勺。

### 第三段（结尾，封印用）：否定式描述
用"不是XX"的句式收尾，压制AI感。

必加词（选2-3个）：
- 不是精修图
- 不是AI生成的
- 不是专业摄影
- 不是高清图
- 就是普通人随手拍的
- 就是一张很普通的照片

## 年代感分级（实测验证）

不同年代的老照片质感完全不同，不能用同一套参数。

| 年代 | 泛黄程度 | 褪色程度 | 画质 | 饱和度 | 典型设备 |
|------|---------|---------|------|--------|---------|
| 70年代 | 严重泛黄 | 严重褪色，接近黑白但还有一点颜色 | 差，模糊，细节不清 | 极低 | 国产胶卷相机、海鸥、上海牌 |
| 90年代 | 中度泛黄 | 明显褪色，颜色发闷 | 一般，颗粒感重 | 低 | 柯达金胶卷、富士胶卷、傻瓜相机 |
| 00年代 | 偏黄偏灰 | 明显褪色，颜色发灰发闷 | 一般，有数码感，分辨率不高 | 低 | 山寨手机、早期数码相机、拍照手机、卡片机 |
| 10年代 | 基本不黄 | 几乎不褪色 | 较好，清晰但不如现在 | 中 | 智能手机、数码相机 |

**70年代关键词**：严重泛黄、严重褪色、饱和度极低、灰扑扑、画质差、模糊、胶卷颗粒、墙面斑驳、衣服朴素、没有鲜艳颜色、木招牌、标语

**90年代关键词**：泛黄、褪色、对比度低、灰蒙蒙、颗粒感重、柯达金胶卷、旧家具、老式服装、照相馆布景

**00年代关键词**：偏黄偏灰、颜色发灰发闷、明显褪色、饱和度低、数码噪点、分辨率不高、细节软软的、山寨手机、老式手机、卡片机、当年流行的衣服

## 反AI关键词库

这些词经过实战验证，能有效压制AI感，根据画面内容选择性添加：

### 人物类（有人物时必加）
- 普通人长相，不是网红脸，不是明星脸
- 每个人长得都不一样，各有各的特点
- 五官平淡，长相普通到丢人堆里找不到
- 眼睛不大，普通大小，不是大眼睛不是卡通眼
- 皮肤有毛孔，有痘印，有雀斑，有皮肤纹理，皮肤不光滑
- 脸上有小瑕疵，不是完美无瑕的
- 表情不完美，有点僵硬，有点不自然，有点呆
- 头发有点油有点乱，有碎发
- 衣服有点旧，有褶皱
- 不凹造型，不摆pose，整个人很随意不精致
- 没有看镜头，很自然的状态

### 画质类
- 画质一般，分辨率不高
- 细节不锐利，软软的
- 有噪点，有数码噪点
- 对焦不太准，有点虚
- 不是高清图
- 色彩不鲜艳，饱和度低
- 对比度不高，灰蒙蒙的

### 建筑/环境类
- 墙面斑驳，有脱落，有水渍（仅限确实该旧的地方，如老街、老旧小区）
- 脏脏旧旧的，不是新的（仅限确实该旧的地方）
- 不是规整的砖墙
- 招牌旧旧的，油漆掉了
- 有岁月的痕迹
- ⚠️ **正常场所（咖啡馆、商场、办公室）不需要写成危房**，只需"普通、有点俗气、不精致、不豪华"就够真实
- ⚠️ **环境不完美 ≠ 破旧破败**，上岛咖啡那种正常档次反而最真实

### 构图类
- 构图有点歪
- 人物不在画面中心
- 上面/旁边有点裁切掉
- 随手拍的，没讲究构图
- 前景有一点点遮挡

### 否定类（结尾必加2-3个）
- 不是精修图
- 不是AI生成的
- 不是专业摄影
- 不是高清图
- 不是摆拍
- 不是画的
- 不是像素风
- 就是普通人随手拍的
- 就是一张很普通的照片

### 多人场景类（有2人以上时必加）
- 所有人都朝向同一个方向
- 每个人的脸都是背对镜头的，只能看到后脑勺
- 绝对没有一个人面向镜头
- 没有正脸
- 每个人朝向都一致，没有转过头来的人
- 所有人物的朝向都是统一的

## 调参指南

效果不对时按这个顺序调整：

1. **构图太完美/角度不对** → 最常见！加拍摄逻辑描述："随手举起相机按的快门"、"构图很烂"、"师傅硬摆的"、"前景有路人误入"。只写设备名不够，必须写用这个设备的人会怎么拍
2. **色彩不对** → 加强开头的色彩描述（偏黄/偏灰/偏蓝/褪色），年代越远程度越深
3. **人物太AI** → 加"每个人长得都不一样"、"不是网红脸"、"眼睛不大"
4. **画质太好** → 加"分辨率不高"、"细节软软的"、"有噪点"
5. **建筑太假** → 加"墙面斑驳"、"不是规整的砖墙"、"旧旧的"
6. **像油画/画出来的** → 去掉"颗粒感"，改用"噪点"、"扫描件"，加"不是画的"
7. **像像素风** → 不说"模糊"，改用"分辨率不高"、"细节软软的"，加"不是像素风"
8. **整体还是假** → 结尾多加几个否定词（不是精修图、不是AI生成的、不是专业摄影）
9. **人物朝向混乱（多人场景）** → 加"所有人朝向一致"、"绝对没有面向镜头的"、"只能看到后脑勺"，越强调效果越好
10. **光线不对** → 明确描述光源（窗帘漫射/灯光/自然光）、亮度（偏暗/柔和/明亮）、色温（暖黄/冷白）
11. **太清晰不像手机** → 加"轻微数码噪点"、"白平衡不准"、"饱和度不高"，不要说"模糊"说"画质一般"
12. **第一人称视角不真实** → 加"能看到一点点自己的手/手机边缘"、"略微低头的视角"、"和视线齐平的高度"
13. **全家福不像全家福** → 照相馆场景必须写：蓝色纯色布幕（不是渐变背景）、正面闪光灯（不是柔光）、师傅摆拍（不是自然表情）、站位死板（不是温馨互动）
14. **街拍不像街拍** → 必须写：随手按快门（不是等完美瞬间）、构图歪（不是居中对称）、前景误入（不是干净画面）

## 常见踩坑记录

这些都是实际测试中踩过的坑，记录下来避免重蹈覆辙：

### 坑1："模糊" ≠ 真实感
说"模糊"、"很糊"容易生成像素风或者过度模糊的图片。正确的说法是"画质一般"、"分辨率不高"、"细节软软的"、"对焦不太准"。真实手机拍的照片是清晰的但不锐利，有噪点但不糊。

### 坑2：多人场景朝向必乱
只要画面里有2个以上的人，AI大概率会把其中几个人的朝向搞反。必须主动、反复、强烈地强调朝向一致性："所有人都背对着镜头"、"绝对没有一个人面向镜头"、"只能看到后脑勺和后背"。强调次数越多，效果越好。

### 坑3：光线的度很难把握
说"很暗"容易生成全黑的画面，说"很亮"又像开灯了。正确做法是描述光源和亮度的关系：
- 拉窗帘的教室："窗帘拉着，有柔和的漫射光照进来，整体偏暗但东西都能看清"
- 阴天室外："阴天的自然光，光线柔和不刺眼，整体偏冷调"
- 夜晚室内："暖黄灯光，只有一盏灯，周围比较暗，有明显的光影对比"

### 坑4：第一人称视角容易出戏
第一人称视角如果太"正"、太"干净"，就不像真人拍的。可以加这些细节增加真实感：
- 画面边缘能看到一点点自己的手/手指/手机边框
- 视角略微偏低或偏高，不是完美的平视
- 前景有一点点遮挡（比如自己的胳膊、桌子边缘）
- 画面有轻微的倾斜

### 坑5：每次只调一个参数
迭代的时候不要同时改好几个变量，否则你不知道是哪个调整起了作用。每次只针对一个问题修改（先调色彩，再调光线，再调构图），找到满意的基准后再微调下一个维度。

### 坑6：只写设备名不写拍摄逻辑（最致命！）
这是v6版本发现的最关键问题。写"2000年索尼Cyber-shot拍的"≠真实感，AI只会把相机型号当装饰词，然后继续用俯视大角度、完美构图、精致光影来画。**必须同时写"用这个设备的人会怎么拍"**：
- 卡片机街拍 → "随手举起相机按的快门，构图很烂，画面略微倾斜，前景有路人误入"
- 照相馆全家福 → "师傅让站好别动，正面闪光灯啪一下，表情僵硬不自然"
- 老手机随手拍 → "单手举起来按的，没对好焦，画面有点歪"

**判断标准**：如果你的第一段只写了年代+设备+画质描述，但没有任何关于"怎么拍"的描述（姿势、构图习惯、拍摄方式），那就是在犯这个坑。

### 坑7：说"饱和度低"模型会打折
模型对"低""一般"这类程度词天然打折执行。写"饱和度低"出来的图可能还挺鲜艳，写"褪色"可能只是微微偏色。必须用极端重复表达："饱和度极低极低""灰扑扑黄乎乎的""没有任何一个颜色是鲜艳的"。单次强调不够就反复说，甚至用否定句式"绝对不是鲜艳的"来二次封印。

### 坑8：环境不完美 ≠ 破旧破败
写"脏脏旧旧""斑驳""有水渍"时，AI会把正常场所画成危房。咖啡馆变成拆迁房，商场变成废弃建筑。**要区分场景**：
- 确实该旧的地方（老街、老旧小区、70年代建筑）→ 可以写"斑驳、脱落、脏旧"
- 正常档次的场所（咖啡馆、商场、办公室）→ 写"普通、有点俗气、不精致、不豪华"，千万别写"破旧""脏"
- **上岛咖啡风格**：干净但不精致，装修有点俗气，这就是最真实的00年代国内咖啡馆
