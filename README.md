# 超真实生图 (Hyperrealistic Image)

> 生成超真实、有生活感的图片。不是"看起来真实"，而是"就是一张普通照片"。

**适用于任何AI图片生成模型**：豆包、Midjourney、DALL-E、Stable Diffusion、Flux、文心一格等。

> ⚠️ **提示**：本技能的提示词在豆包上进行了大量实测优化，在豆包上的出图效果可能会更优秀。其他模型同样适用，但可能需要微调部分参数以达到最佳效果。

## 核心原理

直接写"真实"、"写实"、"照片级"只会触发一种AI风格。要生成真正有真实感的图片，需要用**具体的、可感知的不完美特征**来引导模型。

**真实感 = 具体设备 + 拍摄逻辑 + 画质缺陷 + 不完美细节 + 普通人物 + 反AI否定**

## 黄金三段式提示词结构

1. **第一段（置顶）**：年代 + 设备 + 拍摄逻辑 + 色彩 + 画质
2. **第二段（中间）**：主体内容 + 不完美细节
3. **第三段（结尾）**：否定式描述（封印AI感）

## 支持的风格

| 风格 | 说明 |
|------|------|
| 📱 手机随手拍 | 日常记录、生活vlog、校园生活 |
| 🎞️ 胶片感 | 文艺、复古、氛围感 |
| 📷 纪实街拍 | 街头、人文、故事感 |
| 🏠 生活记录 | 居家、日常、温馨感 |
| 📼 年代旧照片 | 怀旧、70/90/00年代、全家福 |

## 支持的年代分级

| 年代 | 泛黄 | 褪色 | 画质 | 典型设备 |
|------|------|------|------|---------|
| 70年代 | 严重 | 严重 | 差 | 海鸥、上海牌胶卷 |
| 90年代 | 中度 | 明显 | 一般 | 柯达金、富士胶卷 |
| 00年代 | 偏黄 | 明显 | 一般 | 山寨手机、卡片机 |
| 10年代 | 不黄 | 几乎无 | 较好 | 早期智能手机 |

## Files

- `SKILL.md` — Skill instructions (**English**, default entry point). Full methodology with 5 verified examples
- `SKILL_ZH.md` — 技能指令（中文版），针对中国场景/人脸/文化语境优化，含9个实战验证示例
- `main.py` — Python prompt optimizer, auto-detects style/era/realism level

## Usage

### 方式一：作为 Agent Skill 使用
将本仓库的 `SKILL.md` 放入你的 Agent Skill 目录，Agent 会自动读取并按照方法论生成提示词。

### 方式二：直接使用 Python 脚本
```bash
python main.py
```
修改 `main.py` 底部的 `test_inputs` 列表即可生成不同场景的提示词。

### 方式三：直接参考 SKILL.md 写提示词
阅读 `SKILL.md` 中的方法论和示例，手动构建提示词，粘贴到任何AI图片生成工具中使用。
